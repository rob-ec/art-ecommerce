# Art E-Commerce

E-Commerce voltado para artesãos que desejam ampliar sua presença no mercado digital, mas que precisam de uma plataforma simples e acessível. Pensado para artesãos como João, com foco em destacar o valor artesanal de cada peça, evidenciando o cuidado, a história e a autenticidade presentes no processo de criação.

Disponível em [arte-commerce.netlify.app](https://arte-commerce.netlify.app/)

Leia sobre a [Proposta de Projeto](Proposta.pdf)


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
